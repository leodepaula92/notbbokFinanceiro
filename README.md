# notbbokFinanceiro
Este projeto consiste na criação de um caderno temático no NotebookLM com o tema “Educação Financeira Básica: Juros, Inflação e Investimentos Iniciais no Brasil”. A proposta tem como objetivo utilizar a Inteligência Artificial como ferramenta de apoio ao aprendizado, promovendo organização, análise crítica e aprofundamento dos conteúdos estudados.
